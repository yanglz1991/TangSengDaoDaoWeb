# 群管理模块 (GroupManagerModule)

## 概述

群管理模块为飞鸽提供完整的群聊管理功能，只有群主和管理员才能访问这些功能。

## 功能特性

### 1. 群聊邀请确认
- **功能**: 控制群成员邀请好友进群的权限
- **权限**: 群主、管理员
- **说明**: 启用后，群成员需要群主或管理员确认才能邀请朋友进群。扫描二维码进群将同时停用。

### 2. 群主管理权转让
- **功能**: 将群主身份转让给其他群成员
- **权限**: 仅群主
- **说明**: 转让后原群主将自动放弃群主身份

### 3. 成员设置
- **全员禁言**: 启用后，只允许群主和管理员发言
- **权限**: 群主、管理员

### 4. 群成员互加好友控制
- **功能**: 禁止群成员之间互加好友
- **权限**: 群主、管理员

### 5. 群黑名单管理
- **功能**: 管理群黑名单成员，可以添加和移除黑名单成员
- **权限**: 群主、管理员
- **特性**: 
  - 查看当前黑名单成员列表
  - 添加普通成员到黑名单
  - 从黑名单中移除成员
  - 不能将群主和管理员加入黑名单

### 6. 群主、管理员设置
- **功能**: 管理群管理员，可以添加和移除管理员
- **权限**: 仅群主
- **特性**:
  - 查看当前群主和管理员列表
  - 将普通成员提升为管理员
  - 移除管理员权限（不能移除群主）
  - 群主在列表中显示但不能被移除

### 7. 消息设置
- **允许新成员查看历史消息**: 控制新加入的成员是否能查看加入前的聊天记录
- **允许群成员置顶消息**: 控制普通成员是否能置顶消息
- **权限**: 群主、管理员

### 8. 解散群聊
- **功能**: 群主可以解散整个群聊
- **权限**: 仅群主
- **警告**: 解散后所有成员将被移出，聊天记录将被清空，此操作不可恢复

## 技术实现

### 架构设计
- 基于模块化架构，实现 `IModule` 接口
- 使用 `channelManageRegister` 注册群管理功能
- 通过权限检查确保只有合适的用户能访问相应功能

### 权限控制
```typescript
// 群主和管理员权限
if (subscriberOfMe?.role !== GroupRole.owner && subscriberOfMe?.role !== GroupRole.manager) {
    return undefined
}

// 仅群主权限
if (subscriberOfMe?.role !== GroupRole.owner) {
    return undefined
}
```

### 数据源集成
- 使用 `ChannelSettingManager` 管理群设置
- 通过 `WKApp.dataSource.channelDataSource` 调用后端API
- 支持错误处理和用户反馈

### 组件结构
- `ChannelBlacklist`: 群黑名单管理组件
- `ChannelManagerList`: 群管理员列表组件
- `SmallTableEdit`: 可编辑的小表格组件
- `UserSelect`: 用户选择组件

## 使用方式

1. 模块已在主应用中自动注册：
```typescript
WKApp.shared.registerModule(new GroupManagerModule());
```

2. 在群聊设置中，符合权限的用户会看到"群管理"入口

3. 点击进入群管理页面，可以看到相应的管理功能

## 功能详细说明

### 群黑名单管理
- **查看黑名单**: 显示所有被加入黑名单的成员
- **添加黑名单**: 从普通成员中选择并添加到黑名单
- **移除黑名单**: 点击成员旁边的操作按钮移除
- **权限限制**: 不能将群主和管理员加入黑名单

### 群管理员设置
- **查看管理员**: 显示群主和所有管理员
- **添加管理员**: 从普通成员中选择并提升为管理员
- **移除管理员**: 点击管理员旁边的操作按钮移除权限
- **特殊标识**: 群主显示在列表中但不能被移除

## 注意事项

1. 所有群管理功能都需要相应的权限才能访问
2. 功能调用会进行错误处理，失败时会显示错误提示
3. 设置变更会实时刷新页面状态
4. 黑名单和管理员变更会实时同步到服务器
5. 用户选择界面支持搜索和多选（管理员设置支持多选，群主转让仅单选）
